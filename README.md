#“美团点评杯”厦门大学软件开发大赛

##大赛目的
- 将美团大众点评的技术带进校园
- 为美团点评技术俱乐部筛选同学
- 让同学们参与到一线产品的研发中

## 俱乐部简介
- 美团点评技术俱乐部旨在建立一个培养技术人才的平台，将最新的核心技术带到校园，为同学们提供一些需要的学习资料等资源，并定期开展活动，给予俱乐部成员与企业工程师交流学习以及实践实习的机会。
